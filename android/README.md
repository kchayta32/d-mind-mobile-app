# D-MIND Android Workspace

This folder is the active D-MIND mobile workspace. It contains the native Android app, the backend server module, Gradle wrapper files, and implementation reports.

## Main Folders

- `app/`: Android application module with Kotlin Compose UI, Java reliability services, resources, and tests.
- `backend/`: Kotlin/Ktor backend module for API endpoints that need server-side configuration or secrets.
- `docs/`: Markdown and text reports, including folder inventory and migration summary.
- `gradle/`: Gradle wrapper support files.

Root web app files outside this folder are legacy and are not required for Android builds.
